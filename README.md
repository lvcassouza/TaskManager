# **Task Manager App**
Task Manager is an Android application that allows users to manage their daily tasks in an organized and efficient way. It offers features to add, edit, delete, and view tasks, as well as categorize them for easier organization.

## **Features:**

- Task Creation and Editing: Add new tasks with a title and description. Edit existing tasks to update information.
- Categorization: Organize your tasks into different categories (work, personal, studies, etc.).
- List View: See all your tasks in a clear and concise list.
- Task Deletion: Remove completed tasks or those that are no longer relevant.
- Intuitive Interface: Simple and easy-to-use design, based on Jetpack Compose.
- Data Persistence: Uses the Room Persistence Library to store tasks and categories on the user's device.

## **Technologies Used:**

- Language: Kotlin
- Architecture: MVVM (Model-View-ViewModel)
- User Interface: Jetpack Compose
- Data Persistence: Room Persistence Library
- State Management: LiveData
- Coroutines: For asynchronous operations (database)
- Navigation: Jetpack Compose Navigation

## **How to Use:**

1. Clone the repository to your local machine.
2. Open the project in Android Studio.
3. Build and run the app on an emulator or physical device.
4. Add new tasks, edit existing ones, organize them into categories, and mark them as completed.
